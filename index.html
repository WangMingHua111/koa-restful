👍超简单的Restful

Koa Restful 是一个基于 Koa 框架的 Restful Web API 插件开源库，使用 TypeScript 构建。它旨在提供一种轻量、高效、易用的方式来构建 RESTful 风格的后端服务。

GitHub
✨

以简单易用为首要的设计

依赖注入DI、控制器Controller、路由方法Route、参数解析Parameter

🍁

基于Typescript构建

解决javascript弱类型语言不支持反射的问题，助力Node实现DI的能力，来达到和Java、C#等强类型语言类似的体验。

💥

支持依赖注入能力

通过动态（装饰器）或静态（函数）创建依赖，支持属性自动注入（装饰器）或依赖解析（函数）获取注入的依赖实例（对象）。

🐰

推荐配合使用

TypeORM 一个支持各种主流数据库的ORM框架、Art Template 一个简约、超快的模板引擎
